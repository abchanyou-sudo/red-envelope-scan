# 紅包禮金記帳工具

這是一個純前端的網頁應用程式，可以透過手機相機或相簿掃描紅包袋上的姓名和鈔票金額，並將其記錄下來。所有資料都儲存在您的瀏覽器中，並可以隨時匯出成 CSV 檔案。此專案可直接部署在免費的 GitHub Pages 上。

## 主要功能

1.  **OCR 辨識姓名**：使用 Tesseract.js 辨識紅包袋上的中文姓名。
2.  **OCR 計算金額**：辨識多張新台幣鈔票上的面額數字並自動加總。
3.  **資料管理**：新增、刪除、編輯每一筆禮金紀錄。
4.  **CSV 匯出**：將所有紀錄一鍵下載為 CSV 檔案，方便在 Excel 或 Google Sheets 中管理。
5.  **純前端運行**：無需後端伺服器，所有操作都在您的瀏覽器完成，確保隱私。
6.  **手機最佳化**：介面為手機瀏覽器設計，方便在外操作。

## 如何部署到 GitHub Pages

您可以非常簡單地將這個工具部署到您自己的 GitHub 帳號下，完全免費。

**步驟 1: 建立一個新的 GitHub 儲存庫 (Repository)**

1.  登入您的 [GitHub](https://github.com/) 帳號。
2.  點擊右上角的 `+` 號，選擇 `New repository`。
3.  為您的儲存庫命名，例如 `red-envelope-scanner`。
4.  選擇 `Public` (公開)，這樣 GitHub Pages 才能運作。
5.  點擊 `Create repository`。

**步驟 2: 上傳檔案**

1.  在您剛剛建立的儲存庫頁面，點擊 `Add file` -> `Upload files`。
2.  將 `index.html`, `script.js`, `style.css` 這三個檔案拖曳到上傳區域。
3.  點擊 `Commit changes`。

**步驟 3: 啟用 GitHub Pages**

1.  在您的儲存庫頁面，點擊上方的 `Settings` (設定) 標籤。
2.  在左側選單中，點擊 `Pages`。
3.  在 `Branch` 的部分，選擇 `main` 分支，並將資料夾設定為 `/(root)`。
4.  點擊 `Save`。

**步驟 4: 開始使用！**

儲存後，頁面會顯示您的網站網址，格式通常是 `https://<您的GitHub用戶名>.github.io/<您的儲存庫名稱>/`。

請等待幾分鐘讓網站部署完成，之後就可以透過這個網址在您的 iPhone 或任何有瀏覽器的裝置上使用了！

## 使用提示

*   **辨識準確度**：為了獲得最佳的 OCR 辨識效果，請確保在光線充足的環境下拍攝，並盡量讓照片清晰、無模糊。
*   **鈔票掃描**：請將多張鈔票平鋪，避免重疊，並確保每張鈔票上的阿拉伯數字面額（如 1000, 500）都清晰可見。
*   **編輯功能**：表格中的「編號」、「姓名」和「總金額」欄位都可以直接點擊進行修改。
*   **資料儲存**：所有資料都儲存在您當前的瀏覽器中。如果您清除瀏覽器快取或使用不同裝置，資料將不會同步。請記得在完成紀錄後使用 "匯出 CSV" 功能備份您的資料。
